### 1-1. 이 체인이 의미하는 것과 의미하지 않는 것

**의미하는 것**: 해상 리스크는 실재하고, SME가 구조적으로 더 취약하며, 기존 솔루션의 한국 시장 노출이 공개적으로 확인되지 않는다.

**의미하지 않는 것**: "왜 한국 SME가 기존 솔루션을 쓰지 않는가"에 대한 완전한 인과 설명. 이는 1차 인터뷰 없이는 답할 수 없으며, 본 PoC의 범위 밖이다.

> 본 프로젝트는 한국 해상수입 SME가 기존 글로벌 솔루션을 채택하지 않은 이유를 규명하지 않는다. 대신 (1) 검증된 리스크와 구조적 비용열위가 실재하고, (2) 기존 검증 기법(ABC-XYZ, σL 안전재고, 예산제약 최적화)을 이 문제에 적용한 실제 작동 로직을 구현했으며, (3) 이를 한국어·경량·무료로 즉시 쓸 수 있는 형태로 만든 공개 사례가 확인되지 않는다는 사실에 기반한다.

## 2. 차별화 포지셔닝

### 2-1. 기존 기술/솔루션 인정 (필수 — 발표 시 선제적으로 언급)

- IBM 특허 US6078900(1998, 만료, 공공도메인): 예산제약 기반 안전재고 우선순위 배분 알고리즘의 선행기술
- Netstock / Lokad / Flowlity / EazyStock: ABC-XYZ, 동적 안전재고, LT 불확실성 반영 기능을 이미 상용화

**금지 문장**: "기존에 없는 솔루션이다", "세계 최초다", "기존 도구는 이걸 못 한다"

### 2-2. 차별점 — 사실 기반 진술로만 구성

1. 4개 글로벌 솔루션의 한국 지사·한국어지원·한국 도입사례 미확인 (FACT, 검색결과 진술)
2. 위 4개사의 한국어 매체 노출은 Netstock 1건(단순 비교 언급) 외 전무 (FACT)
3. 리스크실재+예산제약+ABC-XYZ+위기배수+예산knapsack+한국정보 6요소 동시조합의 공개사례 미확인 (FACT)

**안전 문구**:
> "공개 한국어 자료 기준으로는 국내 SME 대상 노출·도입 근거가 확인되지 않는다. 이를 곧바로 '시장 공백'으로 단정하기보다, 한국 SME 시장에서의 노출도·현지화·가격 접근성이 충분히 검증되지 않았다는 의미로 해석하는 것이 적절하다."

### 2-3. 우리 아이디어 (핵심 차별화 요소)

**채택: 서비스수준(Service Level) 차등 선택형 다중선택 knapsack**

- 기존 방식: SKU를 "보호함/포기함" 이진 선택으로만 배분
- 신규 방식: SKU마다 90%/95%/99% 등 복수의 서비스수준 옵션을 두고(서비스수준이 높을수록 필요 안전재고·비용 증가), 예산 안에서 SKU별로 어떤 서비스수준을 줄지 동시 최적화하는 **다중선택 배낭문제(multi-choice knapsack)**
- 학술적 근거: 교재 07_SCM_CH5A_Probability_for_Inventory의 cycle service level, Φ⁻¹(분위수함수) 이론을 직접 응용
- 재고비용 구조(자본비용 8~15%, 보관비용 2~5%, 진부화 1~10%, 합계 15~30%)를 손실액 계산에 반영해 시뮬레이션 수치의 임의성 제거(동일 교재 출처)
- 구현 리스크: 외부 API 의존 없음(기존 σL 모듈 확장이라 개발 재사용성 높음)

**보조 기능**:
- "포기 SKU + 예상손실액" 명시적 노출(저비용 UX 추가)
- GSCPI 자동조회 + 사건기반 캘리브레이션(섹션4) — **보조 검증 기능으로 한정, 핵심 차별점 아님**

**폐기**: AI 자연어 진단(아이디어 아님으로 명시적 판단), "실시간 위기지표 자동전환"이라는 표현(GSCPI는 월 1회 갱신이라 실시간 아님)

## 4. 위기배수 캘리브레이션 (사건 기반, 임의 설정 아님)

| 위기등급 | 배수 | GSCPI(검증완료) | 운임지수 보조근거 |
|---|---|---|---|
| Normal | 1.0× | 약 -0.04(2019평균) | Drewry WCI 평균 $1,420/FEU |
| Warning | 1.18× | 2024.1~2월 음수권(GSCPI 단독 부적합 확인됨) | Drewry WCI Shanghai-Rotterdam $4,406~4,661/FEU |
| Crisis | 1.8× | 2021.12 = 4.37(직접검증) | Shanghai-Genoa $12,801/FEU, China-N.Europe $14,407/FEU |
| Extreme | 2.2× | 2022.1~3 = 3.74~2.87(직접검증) | Shanghai-LA $9,112~11,030/FEU |

**핵심 설계 원칙**: GSCPI 단독 매핑은 부적합(홍해위기 시 GSCPI 음수권이나 실제 운임 급등 — 모순 확인됨). **GSCPI + 항로별 컨테이너 운임지수(Drewry WCI) + 사건기반 보정**을 결합해야 함.

**구현 상태**: 고정 시나리오(1.0/1.18/1.8/2.2×)는 개발 완료. GSCPI 자동조회+캘리브레이션 매칭은 **미구현 — 주제 확정 후 착수 예정(이월 항목)**.

**한계(투명 공시 필수)**: 1.18×는 ITF/OECD(홍해위기 +10일, 정시율62%→52%, 운임+130%)로 정밀 검증됨. 그러나 1.8×/2.2×는 "정확히 며칠에서 며칠로 증가했다"는 원출처를 2회 재검증에도 끝내 확인하지 못했다. 이 두 배수는 동시기(2021-22) GSCPI 사상최고치(4.45, 2021.12)와 USITC 운임 데이터(Shanghai-LA $1,503→$10,691, 약7배 폭등)로 "극심한 위기였다"는 정황은 강하게 뒷받침되나, 정밀한 일수 배수 자체는 업계 추정 수준임을 인정한다.

## 6. 발표 시 정직한 한계 공시 (필수 슬라이드)
